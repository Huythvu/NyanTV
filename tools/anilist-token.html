Get your AniList dev token

Run this on your computer — a real desktop browser already clears AniList's Cloudflare, which is the part the TV and the relay can't do. The token you get goes into local.properties as ANILIST_DEV_TOKEN, and the app auto-signs-in with it on every build. AniList tokens last ~1 year, so you do this rarely.

Step 1 — log in & copy the code

Open the AniList login →

After you approve, your browser lands on the pairing callback (it'll show a "token exchange failed" message — ignore that, it's expected). Look at the browser address bar: it now contains ?code=XXXXXXXX…. Copy that code value.

Step 2 — exchange the code (in this browser)

This submits straight to AniList from your browser, riding the Cloudflare clearance it already has. AniList replies with JSON containing access_token.

Step 3 — save it

From AniList's JSON response, copy the access_token value into your local.properties, then rebuild:

ANILIST_DEV_TOKEN=eyJ0eXAiOiJKV1QiLCJhbGci...

That's it — every debug build will now start already signed in to AniList, no QR, no typing, no Cloudflare. Remove the line (and rebuild) to go back to normal login.
